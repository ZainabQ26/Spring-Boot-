## üöÄ Getting Started

### Prerequisites
- Java 11 or higher
- Maven 3.6+
- [Your database if applicable]

### Installation

1. Clone the repository
```bash
git clone https://github.com/zozo080212/Spring-Boot-.git
cd Spring-Boot-
```

2. Build the project
```bash
mvn clean install
```

3. Run the application
```bash
mvn spring-boot:run
```

4. Open your browser and navigate to:
```
http://localhost:8080
```
